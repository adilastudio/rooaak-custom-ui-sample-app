# Support Policy

## Scope

This repository is maintained by Rooaak as a reference starter.

## What we support

1. Bug fixes for documented flows.
2. Security fixes for critical issues.
3. Compatibility updates for Rooaak API/SDK breaking changes.

## Response expectations

- Best-effort support via GitHub Issues.
- No SLA is provided for community support requests.

## Out of scope

1. Custom feature development for individual projects.
2. Provider-specific production ops outside documented starter behavior.
3. Long-term support branches.
